🔄 Pipeline

3-phase automated security testing

📡
Phase 1: Recon
Subdomain enumeration, live host detection, port scanning, tech stack fingerprinting. AI analyzes results and prioritizes targets.
🔍
Phase 2: Vuln Scan
Nuclei vulnerability scanning with AI-powered false positive reduction. CVSS scoring and remediation suggestions.
📊
Phase 3: Report
AI-generated professional pentest reports with executive summary, detailed findings, and prioritized recommendations.

💻 Quick Start

Get started in 3 commands

terminal 
# Install
$ pip install -r requirements.txt

# Configure
$ cp config/example.env config/.env
$ # Edit .env with your API keys

# Run full scan
$ python -m omnisec --target example.com --mode full

✨ Features

Built for real-world security testing

🔄
Multi-Model Orchestration
Automatically routes tasks to the optimal AI model based on task type and model strengths.
🔍
AI False Positive Reduction
AI analyzes scan results to filter out false positives and validate real vulnerabilities.
📊
Professional Reports
Auto-generated pentest reports with executive summary, CVSS scores, and remediation steps.
🎯
Target Prioritization
AI ranks targets by attack surface value — admin panels, APIs, and dev environments first.
🔌
Tool Integration
Works with subfinder, httpx, nuclei, nmap. Graceful fallback when tools aren't installed.
🛡️
Opsec Built-In
Credential vault, clean history, no leak patterns. Security-first development.
